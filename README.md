# wechat-sdk
最全最好用的微信SDK，支持APP，小程序，H5，Web登录支付，企业付款等功能

## 快速开始
以下是APP支付简单例子
```go
wePay := &WePay{
	AppId:     "xxx",
	MchId:     "xxx",
	PayKey:    "xxx",
	NotifyUrl: "xxx",
	TradeType: "xxx",
	Body:      "xxx",
}

results, outTradeNo, err := wePay.AppPay(100))
```
